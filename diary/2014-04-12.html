在Win7下硬盘安装Ubuntu的ISO镜像

方法来自奶牛博客.

Win7 下可以使用 Grub4Dos 或者 Neo Grub (包含在 Easy BCD 中).

1. 配置Grub： 

    title XUbuntu install
    root (hd0,0)
    kernel /vmlinuz boot=casper iso-scan/filename=/linux.iso ro quiet splash
    initrd /initrd.lz
    boot
注意磁盘分区的位置，我的是 hd(0,1)

2. 复制相关文件

  • 将要安装的Ubuntu(各种版本，我使用的是 XUbuntu)镜像文件复制到C:根目录下， 如: xubuntu-13.10-desktop-amd64.iso 复制到 C:\linux.iso;
  • 从此镜像文件中的casper文件夹中解压出vmlinuz及initrd.lz同样放到C:根目录下。

3. 此时重新启动即可看到引导菜单中多了一个 XUbuntu install 的启动项， 启动该项目即可进入到 Live CD 的桌面下了，此时打开终端执行如下指令: 

    sudo umount -l /isodevice

4. 此时就可以使用桌面的安装程序进行安装操作了。

安装完成后如果Windows菜单项不见了，在linux下执行sudo update-grub指令即可。
